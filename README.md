# Malaysia‚Äôs GDP Report Dashboard

An interactive dashboard for analyzing Malaysia‚Äôs **GDP** and **GDP per capita** data across **states**, **sectors**, and **years**.  
Built with **Python**, **Streamlit**, and **Plotly**, it enables users to explore trends, compare states and sectors, and gain insights into Malaysia‚Äôs economic performance.

---

## ‚ú® Features
- **Dynamic KPIs:** Key indicators automatically update based on selected year, sector, and scenario.  
- **Trend Analysis:** Explore GDP and GDP per capita trends across multiple years with dynamic charts.  
- **Correlation Scatter Plot:** Explore the relationship between GDP and GDP per capita  
- **Interactive Choropleth Map:** Visualize GDP or GDP per capita by Malaysian states.  
- **Sector Comparison:** Analyze the contribution of different economic sectors.  
- **State Comparison:** Compare GDP or GDP per capita contributions across states.  
- **Download Options:** Export charts and tables for reports or presentations.

   ```bash
   source myenv/bin/activate
   ```

4. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

5. **Run the dashboard**
   ```bash
   streamlit run app.py
   ```

## üöÄ Usage

1. Once the app launches, open the dashboard in your browser (Streamlit does this automatically).  
2. Use the sidebar to select:
   - **Year** or **Range of Years**
   - **Economic sector** (Agriculture, Manufacturing, Services, etc.)  
   - **Indicator type** (GDP or GDP per capita)  
3. Explore charts, maps, and KPIs for insights.  
4. Download charts and tables for reports or presentations.
